# SnakeGame

to compile and run the project do the following:

A. on unix:
1. run: sudo apt-get install libncurses-dev
2. run: make
3. open SnakeGame

B. on windows: 
1. install a c++ compiler (MinGW is recommended)
2. run: make
3. open SnakeGame.exe

Player 1 commands:
1. up: W
2. down: S
3. left: A
4. right: D

Player 2 commands:
1. up: I
2. down: K
3. left: J
4. right: L
